# ProgressBar

Shows the completion percentage of a task. Use when the operation has a measurable, ongoing duration the user benefits from seeing.

**When to use**: file uploads, exports, multi-step wizards, long-running background jobs.
**When not to use**: short operations (use `Spinner`), layout placeholder (use `Skeleton`).

<Preview>

```tsx
<ProgressBar value={62} max={100} label="Uploading…" showValue />

// Indeterminate (no value)
<ProgressBar label="Processing your export" ariaLabel="Processing" />
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | — | Current value (omit for indeterminate mode) |
| `max` | `number` | `100` | Maximum value |
| `variant` | `'brand' \| 'success' \| 'warning' \| 'error'` | `'brand'` | Color variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Bar thickness |
| `label` | `ReactNode` | — | Visible label above the bar |
| `showValue` | `boolean` | `false` | Show numeric value (e.g. "62%") |
| `ariaLabel` | `string` | — | Accessible label when no visible `label` |

## Accessibility

- Renders with `role="progressbar"` + `aria-valuemin/max/now`
- In indeterminate mode, `aria-valuenow` is omitted (per ARIA spec)
- Always provide either a visible `label` or `ariaLabel`
