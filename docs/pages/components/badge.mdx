# Badge

A small count or label indicator. Use Badge for numeric counts (unread, pending) or for neutral inline status. For order/payment lifecycle statuses, use **StatusBadge** instead.

<Preview>

</Preview>

<Badge label="12" />

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `type` | `'default' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'default'` | Semantic variant |
| `size` | `'sm' \| 'md'` | `'sm'` | Size |
| `label` | `string` | `'Label'` | Text content |

## Badge vs Tag vs StatusBadge

| Use case | Component |
|---|---|
| Numeric count (unread, pending, etc.) | **Badge** |
| Removable filter chip or general label | **Tag** |
| Order / payment / refund lifecycle status | **StatusBadge** |
