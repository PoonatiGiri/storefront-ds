# Skeleton

Animated placeholder shown while content loads. Use Skeleton for layout-preserving loading states — the user sees the shape of the page before the data arrives.

## Default

<Preview>
  <div style={{ width: 240 }}>
    <Skeleton width="100%" height={16} />
  </div>
</Preview>

```tsx
<Skeleton width={240} height={16} />
<Skeleton shape="circle" width={40} height={40} />
<Skeleton shape="text" lines={3} />
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `shape` | `'rect' \| 'circle' \| 'text'` | `'rect'` | Placeholder shape |
| `width` | `number \| string` | — | Width (px when number) |
| `height` | `number \| string` | — | Height (px when number) |
| `lines` | `number` | `1` | Stacked text lines (when `shape='text'`) |
| `noAnimation` | `boolean` | `false` | Disable the pulse |

## Do's and Don'ts

**Do**
- Match skeleton dimensions to real content to avoid jank when content swaps in
- Use `shape="text"` with `lines={n}` for paragraphs — the last line is shortened automatically

**Don't**
- Use a single full-screen skeleton — use a spinner if you can't preserve layout
- Animate when the user has `prefers-reduced-motion` — disable globally for that case
