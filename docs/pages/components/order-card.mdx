# OrderCard

A pre-composed order summary card with status badge, customer, items, total, and contextual primary action. Use OrderCard for compact order lists, recent-order rails, and quick-actions surfaces.

For full order detail with line items and totals, use **LineItem** + **OrderSummary**.

## Status variants

The card's badge and primary action label change based on status:

| Status | Badge | Primary action |
|---|---|---|
| `pending` | warning | Cancel |
| `processing` | info | Track |
| `completed` | success | Reorder |
| `failed` | error | Retry |
| `cancelled` | neutral | View Details |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `status` | `OrderStatus` | _five mandated values_ | Drives badge + action |
| `customerName` | `string` | — | Customer display name |
| `customerInitials` | `string` | — | Avatar initials |
| `orderId` | `string` | — | Order identifier (e.g. "#ORD-1001") |
| `orderDate` | `string` | — | Formatted date string |
| `items` | `OrderItem[]` | — | Line items in the order |
| `total` | `string` | — | Formatted total (e.g. "$134.96") |
| `onPrimaryAction` | `() => void` | — | Primary action click handler |
| `onDetails` | `() => void` | — | Details click handler |
