# Alert

A static, inline banner that communicates status about a section, form, or page. Alerts persist until dismissed ŌĆö they don't auto-disappear.

**When to use**: page-level warnings, form-level errors spanning fields, success confirmations of saved actions.
**When not to use**: transient confirmations like "Saved" (use `Toast`), per-field errors (use `FormField` error state).

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Semantic variant |
| `title` | `ReactNode` | ŌĆö | Heading |
| `icon` | `ReactNode \| false` | _variant-specific_ | Custom icon, or `false` to hide |
| `action` | `ReactNode` | ŌĆö | Inline action (rendered next to body) |
| `onDismiss` | `() => void` | ŌĆö | When provided, shows a close button |
| `dismissLabel` | `string` | `'Dismiss'` | Accessible label for close button |

## Accessibility

- Renders with `role="alert"` so screen readers announce it when it appears
- Dismiss button has a configurable `dismissLabel`
- Icons are decorative ŌĆö variant meaning is conveyed through the title/body
