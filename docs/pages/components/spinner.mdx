# Spinner

An indeterminate loading indicator. Use for inline async states where duration is short or unknown.

**When to use**: button submissions, inline content reloads, anywhere you'd say "loading…".
**When not to use**: layout-heavy loading — use `Skeleton`. Determinate progress — use `ProgressBar`.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Size |
| `tone` | `'brand' \| 'neutral' \| 'inverse'` | `'brand'` | Color tone |
| `label` | `string` | `'Loading'` | Accessible label announced to screen readers |

## Accessibility

- Renders as `role="status"` with `aria-live="polite"` — screen readers announce loading state
- Always pass a meaningful `label` when surrounding context isn't already clear
- The SVG itself is `aria-hidden="true"`; the label is exposed via `sr-only` text
