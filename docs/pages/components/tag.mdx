# Tag

A small inline label for categorizing, filtering, or marking metadata. Tags are non-interactive by default but can be removable.

**When to use**: filter selections, categorical metadata, inline statuses where `Badge` would be too quiet.
**When not to use**: numeric counts (use `Badge`), order lifecycle (use `StatusBadge`).

## Default

<Preview>
  <Tag>Label</Tag>
</Preview>

```tsx
<Tag>Static label</Tag>
<Tag variant="brand" onRemove={() => {}}>Region: EU</Tag>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `'neutral' \| 'brand' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'neutral'` | Visual variant |
| `size` | `'sm' \| 'md'` | `'md'` | Size |
| `iconLeft` | `ReactNode` | — | Icon before the label |
| `onRemove` | `(e) => void` | — | When provided, shows an X remove button |
| `removeLabel` | `string` | `'Remove'` | Accessible label for the remove button |

## Accessibility

- The remove button is a real `<button>` with a configurable `aria-label`
- Non-removable tags are plain `<span>` — they don't appear in the tab order
