# Switch

A binary on/off control for settings that take immediate effect.

**When to use**: settings panels, "activate/deactivate" toggles, feature flags the user controls.
**When not to use**: inside a form requiring submit (use `Checkbox`), mutually exclusive choices (use `Radio`).

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'sm' \| 'md'` | `'md'` | Size |
| `label` | `ReactNode` | — | Label next to the switch |
| `helperText` | `ReactNode` | — | Helper text below the label |
| `labelPosition` | `'left' \| 'right'` | `'right'` | Where the label sits |
| `disabled` | `boolean` | `false` | Native disabled |

## Accessibility

- Renders as a native `<input type="checkbox">` with `role="switch"`
- `Space` toggles, `Tab` moves focus
- Label is linked via `htmlFor`/`id` — clicking the label also toggles
