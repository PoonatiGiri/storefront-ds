# MetricCard

A KPI tile showing a single metric with optional trend indicator. Use MetricCard for dashboards, summary strips, and overview pages.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `title` | `string` | _required_ | Label in the card header |
| `value` | `string` | _required_ | Formatted metric value (e.g. "$24,530") |
| `trend` | `'up' \| 'neutral' \| 'down'` | — | Trend direction |
| `trendValue` | `string` | — | Trend amount (e.g. "12.5%") |
| `period` | `string` | — | Comparison period label |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Visual size |
| `icon` | `ReactNode` | _bar chart_ | Icon in the top-right badge |
